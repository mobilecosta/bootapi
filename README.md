# Jaspe - Chatbot para WhatsApp

Jaspe é um chatbot desenvolvido por Tiago Cunha para ajudar os clientes da Conceitho Tecnologia. Ele utiliza a API do OpenAI para responder às mensagens recebidas pelo WhatsApp.

## Funcionalidades

- Responder às mensagens recebidas pelo WhatsApp utilizando a API do OpenAI.
- Fornecer informações sobre os produtos e serviços da Conceitho Tecnologia.
- Auxiliar os clientes com dúvidas e problemas relacionados aos produtos e serviços da Conceitho Tecnologia.

## Como utilizar

1. Adicione o número do WhatsApp do Jaspe aos seus contatos: +XX XXXXX-XXXX.
2. Envie uma mensagem para o número do Jaspe no WhatsApp.
3. Aguarde a resposta do Jaspe.

## Recursos utilizados

- [OpenAI API](https://openai.com/)

- https://platform.openai.com/api-keys
